nginx:
/usr/sbin/nginx：主程序
/etc/nginx：存放配置文件
/usr/share/nginx：存放静态文件
/var/log/nginx：存放日志


罗列出与nginx相关的软件
dpkg --get-selections|grep nginx

sudo apt-get --purge remove nginx
